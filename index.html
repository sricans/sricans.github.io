Tentang Saya

Hallo nama saya Sri Maulidia. Dulu nama panggilan saya Lidia tapi sejak kelas 1 SMP panggilan saya menjadi Sri. Saya lahir di Bogor pada tanggal 1 Juni 2001. Saya pernah bersekolah di SMP Negeri 7 Bogor di Jln.Paledang no.25 dekat Lapas Bogor. Sekarang saya bersekolah di SMK-SMAK Bogor.Saya sehari-hari bersekolah memakai kerudung tetapi ada sedikit rambut yang terlihat dari kerudung yang saya pakai. Jika saya memakai daleman kerudung saya merasa tidak nyaman karena tidak terbiasa menggunakannya. Jika di lingkungan rumah saya sehari-hari tidak menggunakan kerudung karena saya menggunakannya hanya pada saat bersekolah saja.

riwayat pendidikan saya. Oh iyh saya gini-gini pernah TK loh. TK nya di R.A Wahdatul Ummah tempatnya di dekat rumah saya jadi dulu pas saya TK saya hanya di antarkan saja tidak ditungguin oleh mamah saya. SD saya di SD Negeri Polisi 2 Bogor. Duluuu waktu saya SD saya merasa pintar sampai saya memperoleh NEM tertinggi satu sekolah yaitu 25,50 memang sihh segitu tidak terlalu besar bagi yang lain tapi menurut saya itu merupakan ssuatu kebanggan. Semasa jaman nya SD saya mrupakan cewe terkecil dalam kelas saya karena sampai sekarang saja saya masih merasa kecil ga tumbuh-tumbuh. Waktu SD saya diantar ke sekolah oleh mamah saya sampai-sampai saya dari kelas 1 SD sampai kelas 6 SD ditungguin oleh mamah saya di SD karena waktu itu saya belum mengerti apa-apa.

Selepas SD saya memasuki dunia SMP yang mualai agak sedikit berbeda dari SD. Saya menemui karakter baru,teman baru bahkan sampai guru baru. Di SMP saya masih diantar jemput oleh mamah saya padahal jarak dari rumah ke sekolah tidak terlalu jauh. Di kelas 7 saya menemui 2 sosok cewe yang mengerti banget ke saya. Merekaa bernama Nurinda Utami Padoma dan Yasinthaha Gufrani Arahma. Saya bersahabat dengan mereka kurang lebih 2 tahun. Tapi sewaktu pernaikan kelas 3 SMP saya harus kehilangan mereka berdua karena ada sedikit kesalah pahaman. Ehhh iyh jdi curhat begini wkwkwk. Kita skip ajh yh ke cerita semasa SMA diawal masuk. Duluuuuu banget saya sewaktu masuk SMA saya merasa sendiri karena belum akrab dengan siapa-siapa padahal saya kelas 10 saya sekelas dengan 2 teman satu SMP saya. Lama kelamaan saya menemukan 6 teman yang akrab sekali dengan saya. Mereka bernama Ines Rahmawati Efendi, Dhiya Fakhirah, Najwa Azzahra Siradj, Kahfi Puji Nursetya Ningtyas, dan Shaharani Tiara Putri. Dan inilah foto-foto kebersamaan kita.

Saya mempunyai hobby bermain dengan teman-teman saya karena saya merasa bisa tertawa lepas dengan mereka yh kadang juga nyebelin hahahaha.
